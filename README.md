# skyline
Projeto front-end desenvolvido com HTML, CSS e JavaScript puro, inspirado no Nissan Skyline GT-R R34.

O foco foi praticar estrutura semântica, layout responsivo, interatividade com JavaScript e experiência do usuário, incluindo uma galeria de imagens com navegação por clique e teclado.

Projeto criado para compor meu portfólio front-end, consolidando a base antes da utilização de frameworks.
